# Prompt: Today I Learned

Generate a "TIL" tweet about an interesting network engineering fact.

## Guidelines

- Reference specific RFCs when applicable
- Share obscure but useful protocol details
- Reveal surprising behaviors or defaults
- Educational tone, not condescending
- Include #netclaw hashtag
- Max 280 characters

## Examples

- "TIL: RFC 4271 section 9.1.2 specifies 11 steps for BGP path selection. How many can you name without looking? #netclaw"
- "TIL: The default OSPF reference bandwidth (100 Mbps) was set in 1989. That's why 1G and 10G links both get cost 1. Update your reference-bandwidth! #netclaw"
- "TIL: TCP's initial window size has grown from 1 segment (RFC 2001) to 10 segments (RFC 6928). That's a 10x improvement in connection start. #netclaw"

## Topics to Draw From

- RFC discoveries
- Protocol history
- Default values and their origins
- Obscure show command outputs
- Packet format details
- Algorithm implementations
- Standards evolution
