# Prompt: Achievement

Generate a tweet celebrating a sanitized network engineering accomplishment.

## Guidelines

- DO NOT include real IP addresses (use 192.0.2.x if needed)
- DO NOT include customer or company names
- DO NOT include hostnames from production
- Generalize the accomplishment (e.g., "50 devices" not specific device names)
- Share the win without revealing sensitive details
- Include #netclaw hashtag
- Max 280 characters

## Examples

- "Just finished auditing 50 BGP peers across 3 data centers. Found 2 with missing route filters. Crisis averted before it started! #netclaw"
- "Helped troubleshoot an OSPF adjacency stuck in EXSTART - turned out to be an MTU mismatch on a newly provisioned link. Classic! #netclaw"
- "Analyzed 100 switch configs today. Found 15 still running telnet. Time for some SSH upgrades and hardening! #netclaw"

## Achievement Categories

- Device audits and compliance checks
- Troubleshooting wins
- Automation successes
- Security findings (sanitized)
- Performance improvements
- Migration completions
- Documentation updates

## What to Sanitize

| Real | Sanitized |
|------|-----------|
| Customer name | (omit entirely) |
| IP addresses | 192.0.2.x |
| Hostnames | "core router", "edge switch" |
| Location | "data center", "remote site" |
| Specific counts | Round numbers (50, 100, 200) |
